Mobile MAU Statistics

This past week I pulled some basic mobile MAU stats as preperation for our Localytics integration, and after taking an intial look at hte dataz, I found a handful of insights that I felt were worth noting and passing on to the larger team. I've summed up some of the more interesting findings here for you guys to check out.

The data came from a table called mobile_mau which is currently living in the analytics-db (analytics-db.int.s-cloud.net)

For expensive computations, let's downsample to a size of 50,000

set.seed(1013)
mobile_mau.sample <- sample.df(mobile_mau, 50000)

##Localytics Buckets

likes <- perct(mobile_mau$num_track_likes + mobile_mau$num_playlist_likes, c(0, 5, 15, 40, 100))
ggplot(data = likes, aes(x = as.factor(breaks), y = percent)) + geom_bar(width = 0.7, stat = "identity") + xlab("Number of Likes") + ylab("Percent of Mobile MAU")


followings <- perct(mobile_mau$num_followings, c(0, 5, 15, 40, 100, 250, 600))
ggplot(data = followings, aes(x = as.factor(breaks), y = percent)) + geom_bar(width = 0.7, stat = "identity") + xlab("Number of Followings") + ylab("Percent of Mobile MAU")


followers <- perct(mobile_mau$num_followers, c(0, 5, 15, 40, 100, 400, 1000, 5000))
ggplot(data = followers, aes(x = as.factor(breaks), y = percent)) + geom_bar(width = 0.7, stat = "identity") + xlab("Number of Followers") + ylab("Percent of Mobile MAU")


playlists <- perct(mobile_mau$num_playlists, c(0, 1, 3, 5, 15, 30))
ggplot(data = playlists, aes(x = as.factor(breaks), y = percent)) + geom_bar(width = 0.7, stat = "identity") + xlab("Number of Playlists") + ylab("Percent of Mobile MAU")


reposts <- perct(mobile_mau$num_track_reposts + mobile_mau$num_playlist_reposts, c(0, 5, 15, 40, 100))
ggplot(data = reposts, aes(x = as.factor(breaks), y = percent)) + geom_bar(width = 0.7, stat = "identity") + xlab("Number of Reposts") + ylab("Percent of Mobile MAU")


tracks <- perct(mobile_mau$num_tracks, c(0, 5, 10, 20, 40))
ggplot(data = tracks, aes(x = as.factor(breaks), y = percent)) + geom_bar(width = 0.7, stat = "identity") + xlab("Number of Tracks") + ylab("Percent of Mobile MAU")
